# Amazon US A+ Content 图片尺寸规范

整理自美国站 Seller Central 官方帮助中心及上传界面提示，适用于 Standard 与 Premium A+ 模块。

---

## 1. 基本技术规范

| 属性 | 官方要求 |
|------|-----------|
| 支持格式 | JPG / PNG / BMP（RGB色域） |
| 分辨率 | ≥ 72 dpi |
| 文件大小 | < 2 MB |
| 颜色空间 | RGB |
| 内容限制 | 不得包含价格、促销语、联系方式、二维码等违规内容 |
| 文字叠加 | 手机端可读，避免过多文字 |

---

## 2. Standard A+ Content 图片尺寸

| 模块类型 | 建议尺寸（px） | 备注 |
|----------|----------------|-------|
| 头图 / Banner | 970 × 300 | Standard A+ Header Image |
| 单图模块 | 970 × 600 | 主要展示产品卖点 |
| Logo 图像 | 600 × 180 | 品牌标志 |
| 四图 Highlight 图块 | 220 × 220 | 产品特色展示 |
| 对比图缩略图 | 150 × 300 | 规格 / SKU 对比 |

---

## 3. Premium A+ Content 图片尺寸

| 模块类型 | 建议尺寸（px） | 备注 |
|----------|----------------|-------|
| 大横幅 | 1464 × 600 | Desktop 最小宽度 |
| 单图模块 | 970 × 600 | 与 Standard 相同，可用于 Desktop / Mobile |
| Logo 图像 | 600 × 180 | 品牌标志 |
| 品牌故事模块 | 463 × 625 | 根据模块布局略有差异 |

---

## 4. 设计建议

- 按 **2×尺寸设计再缩小导出** 提高清晰度。
- 确保关键内容在视觉中心区域。
- 图像风格统一，光线、色调、构图协调。
- 文字叠加字体大小适合手机端阅读。
- 保持空白区，避免信息拥挤。
